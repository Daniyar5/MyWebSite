Meme Site - Full-Stack версия

Как запустить проект:

1. Бэкенд (Сервер и База данных)
   - Откройте терминал в папке "backend"
   - Установите зависимости командой: npm install
   - Запустите сервер: npm start (или node server.js)
   - Сервер заработает на порту 3000. Файл базы данных (database.sqlite) создастся автоматически.

2. Фронтенд (Интерфейс)
   - В папку frontend/images/ перенесите ваши картинки из оригинального проекта (bg.jpeg, home.png, meme.png, search.png). Я оставил папки пустыми под ваши оригинальные исходники, чтобы они не потеряли в качестве.
   - Просто откройте файл frontend/index.html в браузере (Google Chrome, Safari, etc.) или используйте Live Server в VS Code.

Теперь вы можете регистрироваться, входить в аккаунт и ставить лайки!
